# 功能清单

## 当前目标

保持默认 Archive 体验和四套非换色系统，同时把 HomeHub 加固为 Cloudflare L2 gateway 与真实上线状态入口。

## 已纳入功能

- Vite + TypeScript 前端工程。
- Three.js 主场景、程序化模型、粒子场和可选 Bloom 后处理。
- Rapier 物理世界，用于重力实验区刚体。
- Lenis + GSAP ScrollTrigger，用于滚动节奏和 section 显影。
- Archive / Nebula / Voyage / Garden 四个独立模式。
- A-F 主模型切换：星图仪、漂浮岛、档案书、宇宙罗盘、黑金花园、能量核心。
- Scroll Gravity：滚动方向、滚动速度、横向 wheel、鼠标和 Space 共同影响重力。
- Launch Constellation：EEI、OpenAIDatabase / MemoryAtlas、PFI、Serenity-Alipay、Archive/nab 共五个整卡入口。
- 每张项目卡明确展示 `L2`、`Live / Deploy-ready / Protected` 与 `L3 gated`；当前 EEI、PFI、Serenity-Alipay、NAB 为已验证 `Live`，MemoryAtlas 为 owner-allowlist Access `Protected`。只有未来未验证项才允许使用公开 `fallbackUrl`。
- 质量分档：`low`、`medium`、`ultra`，支持 `?quality=low|medium|ultra`。
- reduced-motion 降级。
- Cloudflare Workers Static Assets 配置。
- Cloudflare SPA fallback：未知路径回退到 `index.html`。

## 不做范围

- 登录系统。
- 后台 CMS。
- 数据库写入。
- 用户追踪。
- 读取私有原始数据。
- 在项目卡中显示显式 Open/Docs/GitHub 按钮或最后更新时间。
