# 开发记录

## 2026-07-06

### 目标

初始化 `LinzeHomeHub` 工程化前端，并把 v0.3 单文件原型拆成可维护、可部署、可验证的模块。

### 范围

- 当前仓库：`LinzeColin/LinzeHomeHub`
- 本地工作区：当前仓库 checkout。
- 输入原型：用户提供的 `LinzeHomeHub-preview-v0.3.html`。
- 准备稿：本轮准备阶段产出的 `LinzeHomeHub-prep-v0.4.md`。

### 风险与管控

- 视觉风险：四模式容易退化为换色。管控：`modeSystems.ts` 必须定义独立行为参数和 Space 动作。
- 性能风险：Three.js、粒子、Rapier、Bloom 叠加。管控：`qualityProfile.ts` 限制粒子数、物理体数量和后处理。
- 可访问性风险：动效过强。管控：`prefers-reduced-motion` 进入 low 质量档并降低运动。
- 安全风险：误提交私有数据或 secrets。管控：仅使用静态项目数据和公开 fallback URL，构建前做文本扫描。

### 验收标准

- `npm run validate` 通过。
- `npm run build` 通过并生成 `dist/`。
- 本地 preview 可打开，首屏不空白。
- 四个模式、六个模型、滚动重力、Space 引力井、项目跳转可操作。
- `wrangler.jsonc` 指向 `./dist`。

### 当前状态

工程初始化中。
