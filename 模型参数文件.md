# 模型参数文件

## 视觉模式参数

| 模式 | 粒子行为 | 主模型倾向 | Space 动作 | 项目 hover |
|---|---|---|---|---|
| Archive 档案 | 慢速索引线、纸尘、环绕沉降 | 星图仪 / 档案书 | Archive Seal | 索引线点亮 |
| Nebula 星云 | 星云膨胀、中心漏斗、柔雾 | 能量核心 / 星图仪 | Black Hole | 语义星桥发光 |
| Voyage 夜航 | 方向性漂移、航线、雷达扫线 | 宇宙罗盘 / 星图仪 | Warp Jump | 航点偏航 |
| Garden 花园 | 花粉、孢子、上浮、风场 | 黑金花园 / 漂浮岛 | Bloom Pulse | 种子开合 |

## 质量分档

| 档位 | 触发条件 | 粒子数 | 物理体 | 后处理 |
|---|---|---:|---:|---|
| low | reduced-motion、移动端、低性能或 `?quality=low` | 700 | 6 | 关闭 Bloom |
| medium | 默认桌面或 `?quality=medium` | 1500 | 10 | 轻量 Bloom |
| ultra | 高性能桌面或 `?quality=ultra` | 3200 | 16 | Bloom + 更高 DPR |

## 重力公式

```ts
gravityY = clamp(smoothedVelocityY * 0.018 + wheelBoostY, -1.8, 1.8)
gravityX = clamp(pointerBiasX + wheelBoostX, -1.2, 1.2)
energy = decay(max(abs(velocity), abs(wheelDelta), spaceImpulse))
```

## 安全边界

- 不读取 raw data、browser state、cookies、sessions。
- 不提交 `.env`、token、API key。
- `projects.json` 只放公开 URL 和公开说明。

## Cloudflare 上线状态参数

| 参数 | 允许值 | 判定逻辑 |
|---|---|---|
| `compatibilityLevel` | `L2` | 本轮五个 gateway 节点均为 static-first public adapter |
| `deploymentStatus` | `Live` / `Deploy-ready` / `Protected` | `Live` 必须有当前 HTTP 证据；dry-run 只能是 `Deploy-ready`；受 Access 控制可标 `Protected` 但不等于匿名公开 |
| `liveUrl` | 空字符串或已验证 URL | 非空时 `deploymentStatus` 必须为 `Live` 或 `Protected`；`Protected` 还必须说明 Access/allowlist 边界 |
| `fallbackUrl` | 公开 GitHub source URL | `liveUrl` 为空时作为整卡唯一跳转目标 |
| `futureLevel` | `L3 gated` | 动态数据、写入、权限、自动化与外部账户能力不在本轮实现 |

本轮不修改视觉模式、粒子、物理体、Bloom 或重力公式数值。
