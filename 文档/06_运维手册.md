# 运维手册

怎么把它跑起来、参数在哪调、报错了怎么办、都改过些什么。

## 一、怎么跑

改完机器平面，按顺序跑这三条，让人类平面刷新并过关：

```bash
python3 machine/tools/render_human.py            # 先渲染人类平面
python3 machine/tools/check_doc_budget.py        # 体积、中文、纯净三道门
python3 machine/tools/check_blocker_stop.py      # 阻塞重审门
```

## 二、参数在哪调

改一个参数要动哪个文件，都在这。为什么定这个值在 `02_系统架构.md`。

| 参数 | 当前值 | 改哪里 |
|---|---|---|
| 画质档-低 | 700 粒子 / 6 物理体 / 关闭泛光 | `src/app/qualityProfile.ts` |
| 画质档-中 | 1500 粒子 / 10 物理体 / 轻量泛光 | `src/app/qualityProfile.ts` |
| 画质档-高 | 3200 粒子 / 16 物理体 / 泛光加更高像素比 | `src/app/qualityProfile.ts` |
| 重力公式 | 纵横两向重力由滚动速度、横向滚轮、指针与空格键冲量共同 `clamp` 得出 | `src/motion/scrollGravity.ts` |
| 兼容级别 | 二级兼容 | `src/data/projects.json` |
| 上线状态 | 已上线 / 可部署 / 受保护 | `src/data/projects.json` |

## 三、报错了怎么办

| 遇到什么 | 为什么 | 怎么解决 |
|---|---|---|
| 首屏空白或场景不出现 | 三维初始化失败或画质档过高 | 用低画质参数验证；检查浏览器三维支持与 `src/app/qualityProfile.ts` 上限。 |
| `npm run validate` 失败 | 缺必需文件或页面/项目数据契约不符 | 按 `scripts/validate-homehub.mjs` 报错补齐文件与必需字段。 |
| 项目卡指向错误或回退异常 | `projects.json` 的 `liveUrl` 与 `deploymentStatus` 不一致 | 确保 `liveUrl` 非空时 `deploymentStatus` 为已上线或受保护；未验证项用 `fallbackUrl`。 |
| 部署后未知路径 404 | 未配置单页回退 | 确认 `wrangler.jsonc` 的单页回退指向 `index.html`。 |
